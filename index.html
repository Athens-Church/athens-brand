Athens Church Brand Kit

This site is the single source of truth for Athens Church's visual identity. It is designed to be consumed by AI agents and automation tools to produce consistently branded work — correct colors, fonts, logos, voice, and campus assets, automatically. The primary payload is brand-kit.json.

Brand Rules for AI Agents

  • Load brand-kit.json at runtime before generating any Athens Church content. Do not rely on memory.
  • Construct all asset URLs as: base_url + "/" + file
  • For dark backgrounds use alabaster or white logo variants. For light backgrounds use charcoal or cinnabar variants.
  • Typography hierarchy: Bookmania Italic (headings) → Helvetica Neue (subheadings) → Quasimoda (body). Do not substitute licensed typefaces.
  • For campus-specific work, use the appropriate sub_brand logos from sub_brands[].
  • Write in the Athens Church voice: warm, genuine, and accessible. Avoid insider church language. Capitalize He/Him/His when referring to God.
  • All brand rules are enumerated in the rules[] array of brand-kit.json.
